## Getting Started

```bash
npm install @whttlr/ui-core @whttlr/ui-theme
```

```jsx
import { Button } from '@whttlr/ui-core';
import { ThemeProvider } from '@whttlr/ui-theme';

function App() {
  return (
    <ThemeProvider theme="dark">
      <Button variant="primary">Hello World</Button>
    </ThemeProvider>
  );
}
```

## Architecture

The library is built with a modular architecture that allows you to use only what you need:

- **Primitives**: Basic building blocks (Button, Card, Input)
- **Complex**: Advanced components (DataTable, Charts, Navigation)
- **Animated**: Motion-enhanced components
- **Mobile**: Touch-optimized components
- **CNC**: Industrial control components

Browse the stories to explore all available components and their configurations.
